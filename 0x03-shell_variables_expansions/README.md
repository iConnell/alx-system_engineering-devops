# Shell Variables Expansions


### [0-alias](./alias)
```
alias ls='rm *'
creates an alias of the rm command named ls
```


### [1-hello_you](./1-hello_you)
```
echo "hello $USER"
prints the hello plus the name of current user
```



### [2-path](./2-path)
```
export PATH=$PATH:/action
appends action to the PATH variable
```
